RESPONSIVE WEB DESIGN TOPIC

FOR PHONES

For mobile phones which is indeed the smallest version of media queries we can use a minimum width of 320 px, to a maximum width of 480px because they're portrait have a bigger heigth than width

FOR TABLETS AND IPADS

For Tablets and Ipads which are of the landscape version of media queries we can use a minimum width of 481 px, to a maximum width of 767px here because they're relatively wide and have a bigger width than height

FOR SMALL SCREEN LAPTOPS

For Small screen laptops e.g the apple mac Air and macbook pro which are of relatively small laptop screen sizes the version of media queries we can use a minimum width of 769 px, to a maximum width of 1024px here

DESKTOPS, LARGE SCREENS

For desktops and other bigger screen sizes,we can use a minimum width of 1025px and a maximum width of 1200px

FOR VERY LARGE SCREENS, TVs

We can accept that the television in our various home are bigger than our phone and laptop sizes, it is no doubt their media queries should be different 1201px and more
